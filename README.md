# UIMenuDemo
UIMenu usage demo app for iOS 14 and higher.


## Functions
- Create a form using UIScrollView
- Single select form item by using UIMenu (for iOS >= 14.0)
- Single select form item by using UITableView (for iOS < 14.0)
